## "hark link" commands
- Windows
  ```batch
  mklink /h "C:\Users\user\AppData\Local\nvim\init.vim" nvimForWindows
  ```
- Linux
  ```bash
  ln nvimForLinux ~/.config/nvim/init.vim
  ```
## "find file" commands
- Windows
  ```batch
  dir /s [file/dir]
  ```
- Linux
  ```bash
  find -name [file/dir]
  ```
